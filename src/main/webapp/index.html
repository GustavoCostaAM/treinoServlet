Isso é um treino de Servlets

Meu nome é Gustavo Costa, tenho 15 anos e estudo no Instituto J&F. Nos primeiros e segundo anos nós temos um projeto interdisciplinar, onde o segundo semestre do ano é dedicado para o desenvolvimento de um app e um site sobre um projeto criado pelo primeiro ano no qual visa resolver um dos problemas do ODS. Para o ano de 2025, o ODS escolhido é o 9, que tem como pilares a industria, inovação e infraestrutura. O meu projeto é o ScaneIA, aplicativo focado para digitalização de planilhas junto a analises preditivas e prescritivas. Esse treino é dedicado ao ScaneIA, como forma de me familizalizar com as normas de servlets. A visualização do codigo é livre, no entanto, seguimos a licença MIT. Credenciais de acesso estarão salvas em variaveis de ambiente, por segurança, além de deixar o codigo mais versatil para todos que queiram usufruir do treino.
